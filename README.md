## 📝 Описание

FitnessTrackerBot помогает пользователям вести учёт тренировок, отслеживать прогресс и достигать спортивных целей. Бот поддерживает различные виды активности, предоставляет подробную статистику и аналитику, а также напоминает о необходимости регулярных занятий.

### Основные возможности
- ✅ **Регистрация и авторизация** через Telegram-аккаунт
- ✅ **Добавление тренировок** с различными параметрами
- ✅ **Просмотр истории** с фильтрацией
- ✅ **Управление целями** с автоматическим отслеживанием
- ✅ **Статистика и аналитика** по тренировкам
- ✅ **Напоминания** о тренировках
- ✅ **Экспорт данных** в CSV

## ✨ Функциональность

### Добавление тренировок
Пользователь может вносить данные о тренировках с указанием:
- **Тип активности**: бег, ходьба, велосипед, плавание, зал, другое
- **Дата**: в формате ДД.ММ.ГГГГ или "сегодня"
- **Продолжительность**: в минутах (опционально)
- **Дистанция**: в километрах (опционально)
- **Калории**: количество сожжённых калорий (опционально)
- **Заметки**: комментарии о погоде, самочувствии (опционально)

### Статистика и аналитика
- **Общее количество тренировок** за период
- **Суммарная дистанция** (км)
- **Суммарное время** (минуты)
- **Суммарные калории**
- **Средние показатели** на тренировку
- **Фильтрация** по видам активности
- **Сравнение периодов** (неделя/месяц)

### Цели
- Установка целей на определённый срок
- Автоматическое отслеживание прогресса
- Уведомления при достижении цели
- Поддержка целей по дистанции, времени, количеству тренировок

## 🛠 Технический стек

- **Язык программирования**: Python 3.10+
- **Библиотеки**:
  - `aiogram` 3.x - работа с Telegram API
  - `aiosqlite` - асинхронная работа с SQLite
  - `pydantic` - валидация данных
  - `datetime` - работа с датами
- **База данных**: SQLite3
- **Архитектура**: Многослойная (репозитории, сервисы, хэндлеры)
